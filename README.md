- 🔍 Search Issues
- 📦 Modal View for Details

## 📚 Questions & Answers

### 1. Difference between var, let, const
- `var` → function scoped  
- `let` → block scoped  
- `const` → cannot be reassigned  

### 2. Spread Operator (...)
Used to copy or merge arrays/objects.

### 3. map vs filter vs forEach
- map → returns new array  
- filter → returns filtered array  
- forEach → no return
